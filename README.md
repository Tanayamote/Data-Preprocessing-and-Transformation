# Data-Preprocessing-and-Transformation
A Python Data Wrangling project demonstrating missing value handling, outlier detection using IQR method, log transformation, normalization, and standardization on an academic performance dataset using Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.

#  Data Wrangling II – Outlier Detection & Data Transformation

##  Project Overview
This project demonstrates advanced Data Wrangling techniques using Python on an Academic Performance dataset.

The notebook includes:
- Handling missing values
- Detecting outliers using IQR method
- Outlier treatment
- Logarithmic transformation
- Min-Max Normalization
- Standardization
- Data visualization using boxplots and histograms

---

##  Features
- Importing and analyzing dataset
- Missing value treatment
- Outlier detection using IQR
- Replacing outliers with median values
- Log transformation of data
- Min-Max Scaling
- Standardization using StandardScaler
- Data visualization with Matplotlib and Seaborn
